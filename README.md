# os-test

A simple joke-telling application written in Node.js.

## Features

- Random joke generation
- Collection of programming and general jokes
- Command-line interface
- Easy to use

## Installation

No installation required! Just ensure you have Node.js installed.

## Usage

### Tell a Random Joke

```bash
node jokes.js
```

Or using npm:

```bash
npm run joke
```

### Show All Jokes

```bash
node jokes.js --all
```

Or using npm:

```bash
npm run jokes
```

### Run Tests

```bash
npm test
```
